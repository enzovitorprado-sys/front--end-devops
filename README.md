# git comandos
- git clone (clona um repositorio remoto do github para o pc)
- git status (exibe o status do repositorio local)
- git add nome-arquivo(adiciona no stage changes o arquivo)
- git add . (adiciona todos os arquivos no stage changes)
- git checkout -b (cria uma nova branch)
- git commit -m (salva os arquivos no repositorio local)
- git push (enviar o repositorio local para o repositorio remoto)
